# Executable Hermes Usage Guide

## Overview

**Hermes** is a static analysis tool built on LLVM , designed for program analysis at the intermediate representation (`.bc`) level.



## Environment

Hermes is pre-installed in our Docker image, which is available at:

https://drive.google.com/file/d/1xnokXJqA_Ij0hJCc3vYQUXwmEH6xqMHX/view?usp=sharing

The executable path is:

```bash
/hermes/hermes_build/pp-check
```

The Docker image also includes the complete experimental dataset used in our evaluation, allowing all experiments to be reproduced without requiring any additional data downloads.



## Input Requirements

Hermes accepts **LLVM IR bitcode files** (`.bc`) as input. You can generate them using `clang`.



## Basic Command

To run Hermes, use the following command:

```bash
/hermes/hermes_build/pp-check \
  --execution-mode=normal \
  --ps-npd \
  --psa-enable-side-effect-source \
  --psa-retsmry-track-depth=3 \
  --segsolver-interface-region-max-depth=3 \
  --psa-enable-global-var-in-trace=false \
  -falcon-cg \
  -nworkers=10 \
  -report=/path/to/your_report \
  /path/to/your.bc

```



## Optional Parameter

```bash
--psa-enable-arg-symbol
```

Enables symbolic handling of function arguments.



## Output

Hermes generates an analysis report under the specified path.



## Experiment Reproduction

To reproduce our experiments, simply execute the following script:

```shell
/hermes/run.sh
```

This script automatically runs the complete Hermes analysis pipeline, including vulnerability detection and post-processing deduplication.

After execution, the deduplicated vulnerability reports are generated for each subject project and can be found at:

```shell
/hermes/hermes_subject/<project_name>/bug_reports/<checker_type>/deduplicate/
```

Each `deduplicate` directory contains the final deduplicated vulnerability reports produced by our tool.

In addition, the `log` directory records the runtime and memory usage statistics of the tool during execution, which can be used for performance analysis and experimental evaluation.
